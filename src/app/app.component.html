Sesame Home

A mobile application built to connect clients and local business, providing the most efficient online solution for the business across GTA area. Whether you’re looking for a place to live or services nearby.

Technology

Sesame Home is built with React Native. Integrating with the Google Maps API for business and service lookup. A PHP server host on AWS acts as a middle-man, handling backend logic and database updates. Sentry is for product error tracking.

Gamie.io

A Nintendo-like multi-player gaming web application.

Technology

Gamie.io is built with Angular2+, and uses Bootstrap for a mobile-first layout. It uses an Express/Node.js app on Heroku to handle user accounts and authentication and stores data at MongoDB.

Ableoo

A web application provides online booking for on-site phone repair service.

Technology

Ableoo is built with Java SpringMVC and using MySQL as database connected by JDBC. The application is implemented by Apache Tomcat and host on Amazon Web Server.
